# Advanced-Information-Processing
VEGAで使用しているベクトルやUART通信関連関数です。コメントを残しての使用をしてください。複数ファイルを一つにまとめているため、エラーが出る可能性があります。問題点がある場合は fox.vega.soft@gmail.com 宛に報告をお願いします。


These are the vector and UART communication functions used at VEGA. Please keep the comments when using them. Since multiple files are combined into one, errors may occur. If you encounter any problems, please report them to fox.vega.soft@gmail.com.
